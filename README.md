# Goland Cheatsheet

A place to keep neat, cool, secret GoLand shortcuts currently for Mac only.

## Tool Windows/Popups

|Action|Shortcut|
|------|--------|
|Project|⌘1|
|Version Control|⌘9|
|Run|⌘4|
|Debug|⌘5|
|Terminal|⌥F12|
|Editor|Esc|
|Navigation Bar|⌘↑|
|Hide all tool windows|⇧⌘F12|

## Code Completion

|Action|Shortcut|
|------|--------|
|Basic completion|⌃Space
|Smart Completion|⌃⇧Space
|See suggested parameters for method/function|⌘P
